# 🖥️ Arquitetura de Computadores - TADS IFRN

Este repositório contém a resolução de 5 listas de exercícios práticos desenvolvidas na linguagem **Assembly MIPS**, utilizando o simulador **MARS**. As atividades fazem parte da disciplina de Arquitetura de Computadores do curso de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) no Instituto Federal Campus Natal-Central.

## 🛠️ Ferramentas e Requisitos

* **ISA:** MIPS32.
* **Simulador:** [MARS](https://dpetersanderson.github.io/download.html) (MIPS Assembler and Runtime Simulator).
* **Java:** Necessário para rodar o simulador `.jar`.

---

## 🚀 Como Executar

1.  Abra o simulador MARS.
2.  Carregue o arquivo `.asm` referente à questão desejada.
3.  Utilize a tecla **F3** para realizar o *Assemble*.
4.  Utilize a tecla **F5** para rodar o programa (*Run*).
